# Changelog

Notable changes to the template. **Already using it?** Updates never touch *your* generated files (PROCESS.md, DATA_CHECKLIST.md, reports) — they're yours. To pick up template improvements, re-download the files you use (prompts, seed reference) or `git pull` if you cloned.

## 1.2 — July 2026
- **Quick Physical overhauled:** embedded 2026 fact card + benchmark thresholds (works well with zero attachments), 8-item vitals panel (🟢/🟡/🔴), stage-specific checks, spending-sensitivity line in the projection, and a `QUICK PHYSICAL SUMMARY` save-block enabling year-over-year comparison even on the lite tier.
- Added `chat/QUICK_PHYSICAL_WORKSHEET.md` — 10-minute pre-session fill-in form.
- Added a fictional example Quick Physical output in `examples/`.
- Added `MAINTENANCE.md` (annual seed-refresh ritual) and this changelog.
